A CSS pseudo-class is a keyword added to a selector that lets you style an element based on its specific state or condition. Unlike standard classes, the browser applies them automatically when certain conditions are met , such as when a user hovers over a link or an input is invalid. SyntaxPseudo-classes always start with a single colon (:)
