1 - To create HTML boiler plate, click the exclamation point then tab.

2 - To apply CSS using the ID property of elements, in the external css document, start with a # symbol. Other ways include using the class or general elements like the body.

3 - Going forward learn to name the default landing page of my Web pages defaults to index.html so as to imply its a start page, ie the HOME Page.

4 - There're 3 methods of applying CSS:

Inline
Properties are set in the opening tags of the elements by setting the setting the style attribute
External
A link on the Index file is made to or linked to the external folder
Internal
Relies on the use of a pair of style tags that are placed within the header section of the said html file.

5 - In CSS, the border-width, border-style and border-color properties can be merged into one line as opposed to having 3 separate lines for each. hence, it would be; border:3X Solid rgb(245,245,234)

6 - An inner border property of an element is called an inset while the outer border property is called outset.

7 - When calculating the height or width of an element say a Div element that we are allocating, we can disregard the padding and border width values by including the box-sizing: border-box property.

8 - To select all elements on the webpage, use the asterix character followed space then by the opening and closing brackets as below. * { }
